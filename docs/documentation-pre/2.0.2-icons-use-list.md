# 2.0.2 Icons In Use

Complete list of icons used in the Kolkrabbi Editor. All app icons are now centralized in `src/components/icons/svg/app-icons/`.

## Icon Naming Convention

Icons follow the pattern: `{original-name}-{tool/ui}-{category}.svg`

Examples:
- `pointer-selector-tool-select.svg` - Pointer icon for select tool
- `caret-down-ui-dropdown.svg` - Caret icon for UI dropdowns
- `rectangle-tool-shape.svg` - Rectangle icon for shape tool

## Usage

All icons use the `app-icons` folder:

```jsx
<Icon name="pointer-selector-tool-select" folder="app-icons" size={20} />
<Icon name="caret-down-ui-dropdown" folder="app-icons" size={12} />
```
