## Quick Reference

- **Icon Component**: `src/components/icons/Icon.jsx`
- **SVG Storage**: `src/components/icons/svg/`
- **Folder Convention**: Icons organized by first letter or custom folders (e.g., `tools-name/`, `a1/`)

## Usage

```jsx
import Icon from '@/components/icons/Icon'

// Auto-folder (uses first letter)
<Icon name="star" size={24} />

// Custom folder
<Icon name="rectangle" folder="tools-name/shape-align" size={20} />
```
