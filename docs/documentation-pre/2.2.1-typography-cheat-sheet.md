### Right Grotesk Mono Weight Scale

Right Grotesk Mono by Pangram Pangram uses a non-standard weight scale:

| Weight Name | CSS Value | Usage in System |
|-------------|-----------|-----------------|
| Fine | 100 | `.kol-helper-fine-*`, `.kol-mono-*-fine` classes |
| Light | 200 | Reserved for future use |
| Regular | 340 | Reserved for future use |
| Medium | 470 | Standard helper/mono classes |
| Bold | 620 | Reserved for future use |
| Dark | 770 | Reserved for future use |
| Black | 900 | Reserved for future use |

**Currently Used:** Fine (100) and Medium (470)

## Responsive Scaling

All classes except Helpers use `clamp(min, preferred, max)`:

```css
/* Example: kol-text-md */
font-size: clamp(14px, 1.5vw, 18px);
```

- **min**: Mobile size (default)
- **preferred**: Viewport-relative fluid value
- **max**: Desktop size (â‰¥1024px)

**Helper classes use fixed sizes** for consistent button/control dimensions.

## Color Token

All typography classes use:

```css
color: var(--kol-surface-on-primary);
```

Text automatically adapts in `.bg-surface-inverse` contexts. Pair with `.text-auto` utility for explicit semantic color control.
