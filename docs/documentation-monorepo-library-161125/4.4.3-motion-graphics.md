# 4.3.3 Motion Graphics Collection

## Overview
The Motion Graphics collection showcases experimental motion graphics, generative animations, and Touch Designer explorations demonstrating dynamic visual storytelling.

**Route:** `/collections/motion-graphics`
**Data Location:** Inline in `/apps/web/src/routes/collections/MotionGraphics.jsx`
**Total Items:** 9 videos

## Data Structure

```javascript
{
  id: number,
  title: string,                   // Video title
  subtitle: string,                // Brief description
  category: string,                // Primary category
  type: string,                    // Animation type
  year: string,                    // Year created
  description: string,             // Detailed description (for modal)
  thumbnailUrl: string,            // Thumbnail image path
  videoUrl: string,                // Video file path
  featured: boolean,               // Featured status
  touchDesigner: {                 // Optional Touch Designer metadata
    patchName: string,
    version: string,
    resolution: string,
    fps: string,
    operators: string[],
    notes: string
  }
}
```

## Motion Graphics Types

Motion graphics are categorized by the following types:

### By Technique
- **`Generative`** - Procedurally generated animations
- **`Simulation`** - Physics/particle simulations
- **`Abstract`** - Non-representational motion
- **`Morphing`** - Shape transformation animations
- **`GLSL`** - Shader-based animations
- **`Particle`** - Particle system animations
- **`Fluid`** - Fluid dynamics simulations

### By Style/Subject
- **`Kinetic`** - Kinetic typography/motion
- **`Data`** - Data visualization animations
- **`Visualization`** - Information visualization
- **`Reactive`** - Audio-reactive animations
- **`Pattern`** - Pattern-based animations
- **`Typography`** - Type-focused animations

### By Application
- **`Shader`** - Custom shader experiments
- **`Audio`** - Audio-visual work
- **`Typography`** - Text in motion

## Touch Designer Integration

Videos can include Touch Designer metadata displayed in the video player modal:

```javascript
touchDesigner: {
  patchName: 'geometric_patterns_v01.toe',
  version: '2023.11880',
  resolution: '1920x1080',
  fps: '60',
  operators: ['Noise TOP', 'Feedback TOP', 'Displace TOP'],
  notes: 'Uses feedback loop with displacement for organic movement'
}
```

When present, a "TD" pill badge is displayed on the card.

## Video Playback

### Hover Preview
Videos play automatically on hover using the `VideoThumbnail` component:
- Video plays on mouse enter
- Pauses and resets on mouse leave
- Uses `preload="metadata"` for performance

### Modal Player
Clicking a video opens a full-screen modal with:
- Full video player with controls
- Video title and subtitle
- Description
- Touch Designer metadata (if available)
- Category/type/year tags
- Close button

## Filtering

Motion graphics can be filtered by:
- **Category** - Primary classification
- **Type** - Technique, style, or application (see types above)
- **Year** - Year created

## Display Behavior

- **Initial Load:** Shows all 9 videos (no load more needed)
- **Aspect Ratio:** Videos use 16:9 aspect ratio (`aspect-video`)
- **Card Style:** Same hover overlay as illustrations/logomarks
- **Empty State:** "No motion graphics match your current filters" with clear filters option

### VideoThumbnail Component
Custom component handling hover-to-play functionality:
```jsx
const VideoThumbnail = ({ videoUrl, thumbnailUrl, alt, isHovered }) => {
  const videoRef = useRef(null)

  useEffect(() => {
    if (videoRef.current) {
      if (isHovered) {
        videoRef.current.play()
      } else {
        videoRef.current.pause()
        videoRef.current.currentTime = 0
      }
    }
  }, [isHovered])

  return (
    <video
      ref={videoRef}
      src={videoUrl}
      poster={thumbnailUrl}
      muted
      loop
      playsInline
      preload="metadata"
      className="w-full h-full object-cover"
    />
  )
}
```

## Best Practices

### Adding New Videos

1. Add video file to `/public/videos/motion-graphics/`
2. (Optional) Add thumbnail to `/public/videos/motion-graphics/thumbnails/`
3. Add entry to `videos` array in `/apps/web/src/routes/collections/MotionGraphics.jsx`:

```javascript
{
  id: 10,
  title: 'Your Video Title',
  subtitle: 'Brief description',
  category: 'Generative',
  type: 'Abstract',  // See types list above
  year: '2024',
  description: 'Detailed description for modal',
  thumbnailUrl: null,  // or '/videos/motion-graphics/thumbnails/thumb.jpg'
  videoUrl: '/videos/motion-graphics/motion-graphic-10.mov',
  featured: false,
  touchDesigner: {  // Optional
    patchName: 'your_patch.toe',
    version: '2023.11880',
    resolution: '1920x1080',
    fps: '60',
    operators: ['Operator 1', 'Operator 2'],
    notes: 'Technical notes'
  }
}
```

### Type Selection Guidelines

- Choose the **primary technique** first (Generative, Simulation, etc.)
- Use type to describe the **visual result** (Abstract, Pattern, etc.)
- For Touch Designer work, include full metadata in `touchDesigner` object
- Be consistent within similar animation styles

### Video Format Recommendations

- **Resolution:** 1920x1080 (Full HD)
- **Format:** .mov or .mp4
- **Codec:** H.264 for web compatibility
- **Frame Rate:** 30fps or 60fps
- **Duration:** 5-30 seconds for preview loops
- **File Size:** Optimize for web (<10MB recommended)
