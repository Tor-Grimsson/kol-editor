# 4.4.0 Foundry Pages

## Overview

The Foundry section provides access to typeface specimens, font information, and licensing. It serves as the main entry point for exploring Kolkrabbi's type design work.

## Main Foundry Pages

### FoundrySpecimens (`/foundry/specimens`)

**Purpose**: Landing page for the type specimen library with filtering capabilities

**Location**: `apps/web/src/routes/foundry/FoundrySpecimens.jsx`

**Key Features**:
- OverviewHero with badge, title, and category pills
- FeaturedItemsCarousel showing 5 specimen previews with live card renders
- SpecimenFilters component with three view modes and filtering
- Grid display of TypefaceCard components
- Bottom CTA linking to featured specimens

**Components Used**:
- `OverviewHero` - Hero section with badge and categories
- `FeaturedItemsCarousel` - Rotating specimen previews
- `SpecimenFilters` - Filter UI with view mode toggle
- `TypefaceCard` - Individual specimen/hub cards
- `FoundryCTA` - Bottom call-to-action

**Data Source**: `apps/web/src/data/foundry/specimens.js`

### Specimen Data Structure

**File**: `apps/web/src/data/foundry/specimens.js`

**Exports**:
1. `specimenHubs` - 7 typeface collection landing pages
2. `allSpecimens` - 15+ individual specimen pages
3. `allSpecimenData` - Combined export for filtering

**Specimen Properties**:
```javascript
{
  id: 'unique-id',
  name: 'Display name',
  typeface: 'TG Typeface Name',
  fontFamily: 'CSSFontFamily',
  fontStyle: 'italic', // optional
  subtitle: 'Pattern count or status',
  description: 'Description text',
  link: '/specimen/path',
  category: 'Editorial' | 'Poetry' | 'Systems' | 'Layout' | 'Display' | 'Preview',
  hub: 'parent-hub-id', // for specimens only
  type: 'hub' | 'specimen'
}
```

## Filtering System

### SpecimenFilters Component

**Location**: `apps/web/src/components/sections/foundry/SpecimenFilters.jsx`

**View Modes**:
1. **Specimen Hubs** - Shows 7 typeface collection pages (default)
2. **All Specimens** - Shows all 15+ individual specimen pages
3. **By Typeface** - Filter specimens by typeface and/or category

**Filter Options**:
- **Typeface**: TG Málrómur, TG Gullhamrar, TG Dylgjur, TG Silfurbarki, TG Rót, TG Orðspor, TG Tröllatunga
- **Category**: Editorial, Poetry, Systems, Layout, Display, Preview

**Features**:
- Expandable/collapsible with filter icon button
- Active filter count display
- "Clear all" button when filters are active
- Real-time filtering with useMemo optimization

## Featured Specimens Carousel

**Configuration**:
- Auto-rotation: 5 seconds
- Live card previews scaled to 50%
- Background: `bg-surface-inverse` with padding
- Shows typeface name in pill badge

**Featured Specimens** (5 total):
1. Málrómur Complete Selection (Editorial)
2. Gullhamrar Icelandic Poetry (Poetry)
3. Dylgjur Complete Selection (Editorial)
4. Silfurbarki Complete Selection (Preview)
5. Orðspor Layout Grid Systems (Layout)

**Preview Card Imports**:
- Uses actual specimen card components
- Scaled down 50% with CSS transform
- Positioned with `top left` origin and padding
