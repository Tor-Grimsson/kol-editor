# 4.3.1 Illustrations Collection

## Overview
The Illustrations collection showcases visual explorations and conceptual illustration work across various mediums, styles, and subjects.

**Route:** `/collections/illustrations`
**Data File:** `/apps/web/src/data/illustrations.js`
**Total Items:** 40 illustrations

## Data Structure

```javascript
{
  id: number,
  name: string,                    // Display name
  description: string,             // Brief description
  illustrationName: string,        // Asset name for <Illustration> component
  category: string,                // Primary category
  type: string,                    // Illustration type/style
  year: string                     // Year created
}
```

## Illustration Types

Illustrations are categorized by the following types:

### By Medium/Technique
- **`Digital`** - Digital painting/illustration
- **`Vector`** - Vector-based illustrations
- **`3D`** - 3D rendered illustrations
- **`Mixed Media`** - Combining multiple techniques

### By Style
- **`Abstract`** - Non-representational, geometric forms
- **`Minimalist`** - Clean, simple forms
- **`Isometric`** - Technical isometric illustrations
- **`Flat`** - Flat design style
- **`Textured`** - With texture/grain effects

### By Subject/Purpose
- **`Editorial`** - Magazine/article illustrations
- **`Concept`** - Concept art, visual development
- **`Character`** - Character design/mascots
- **`Technical`** - Diagrams, instructional
- **`Pattern`** - Repeating patterns, textiles
- **`Icon`** - Icon illustrations
- **`Product`** - Product visualization
- **`Portrait`** - Portrait illustrations
- **`Landscape`** - Environmental scenes

### By Genre
- **`Fantasy`** - Fantasy/mythical themes
- **`Sci-Fi`** - Science fiction themes
- **`Nature`** - Organic, natural subjects
- **`Urban`** - City/architectural themes


### Brand Asset Illustration Types:

  Product-Focused

  - Product - Product visualization/showcase
  - Feature - Specific feature illustrations
  - UI - User interface mockups/illustrations
  - Hero - Large header/hero section illustrations
  - Spot - Small decorative accent illustrations

  Purpose/Context

  - Marketing - General marketing/brand materials
  - Explainer - How-to, step-by-step guides
  - Onboarding - App/product onboarding flows
  - Empty State - No-content placeholder illustrations
  - Landing Page - Landing page visuals

  Conceptual

  - Metaphor - Visual metaphors for features/concepts
  - Diagram - Flowcharts, process diagrams
  - Infographic - Data/information visualization

## Filtering

Illustrations can be filtered by:
- **Category** - Primary classification
- **Type** - Medium, style, or subject (see types above)
- **Year** - Year created

Filters use the format: `"type:Digital"`, `"year:2024"`, etc.

## Display Behavior

- **Initial Load:** Shows 9 illustrations
- **Load More:** "Show all 40 illustrations" button appears if more than 9 items
- **Filter Reset:** Returns to 9-item display when filters change
- **Empty State:** "No illustrations match your current filters" with clear filters option

## Best Practices

### Adding New Illustrations

1. Add SVG file to `/packages/ui/src/atoms/illustrations/svg/`
2. Import in `/packages/ui/src/atoms/illustrations/Illustration.jsx`
3. Add entry to `/apps/web/src/data/illustrations.js`:

```javascript
{
  id: 41,
  name: 'Your Illustration Name',
  description: 'Brief description of the work',
  illustrationName: 'your-illustration-name',  // matches SVG filename
  category: 'Conceptual',
  type: 'Digital',  // See types list above
  year: '2024'
}
```

### Type Selection Guidelines

- Choose the **most defining characteristic** of the illustration
- For multiple applicable types, prioritize: Medium > Style > Subject > Genre
- Be consistent within similar works
- Update filter options if new types are needed
