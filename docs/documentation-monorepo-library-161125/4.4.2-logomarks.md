# 4.3.2 Logomarks Collection

## Overview
The Logomarks collection showcases brand identity experiments and logomark designs exploring form, function, and symbolism across diverse industries and applications.

**Route:** `/collections/logomarks`
**Data File:** `/apps/web/src/data/logomarks.js`
**Total Items:** 25 logomarks

## Data Structure

```javascript
{
  id: number,
  name: string,                    // Brand/project name
  description: string,             // Brief description
  logoName: string,                // Asset name for <Logomark> component
  category: string,                // Primary category
  type: string,                    // Logomark type
  year: string                     // Year created
}
```

## Logomark Types

Logomarks are categorized by the following types:

### By Form
- **`Symbol`** - Symbolic/iconic mark
- **`Wordmark`** - Text-based logo (gets special sizing: 160px vs 96px)
- **`Lettermark`** - Initial/letter-based mark
- **`Monogram`** - Combined letter design
- **`Abstract`** - Abstract geometric form
- **`Pictorial`** - Literal/pictorial representation
- **`Emblem`** - Badge/seal style
- **`Combination`** - Symbol + text combination

### By Industry
- **`Tech`** - Technology/software
- **`Creative`** - Design/creative studios
- **`Food & Beverage`** - Restaurants, cafes, food brands
- **`Retail`** - Retail/e-commerce
- **`Healthcare`** - Medical/wellness
- **`Finance`** - Financial services
- **`Education`** - Educational institutions
- **`Nonprofit`** - Nonprofit organizations

### By Style
- **`Minimal`** - Minimalist design
- **`Geometric`** - Geometric construction
- **`Organic`** - Natural, flowing forms
- **`Modern`** - Contemporary style
- **`Vintage`** - Retro/vintage aesthetic
- **`Playful`** - Fun, approachable
- **`Elegant`** - Sophisticated, refined
- **`Bold`** - Strong, impactful

## Categories

Primary categories for organizing logomarks:
- Brand Identity
- Corporate
- Startup
- Product
- Personal

## Filtering

Logomarks can be filtered by:
- **Category** - Primary classification
- **Type** - Form, industry, or style (see types above)
- **Year** - Year created

## Display Behavior

- **Initial Load:** Shows 9 logomarks
- **Load More:** "Show all 25 logomarks" button appears if more than 9 items
- **Filter Reset:** Returns to 9-item display when filters change
- **Empty State:** "No logomarks match your current filters" with clear filters option

## Special Handling

### Wordmark Sizing
Logomarks with `type: 'Wordmark'` receive special sizing in the CollectionCard component:
- **Wordmarks:** 160px width
- **All other types:** 96px width

This is handled automatically by the CollectionCard component:
```javascript
const logoSize = item.type === 'Wordmark' ? 160 : 96
```

## Collections Feature

The logomarks data file includes a `logomarkCollections` export for grouping related marks:

```javascript
export const logomarkCollections = [
  {
    title: 'Tech Startups',
    count: 8,
    items: ['SaaS', 'AI', 'Mobile']
  },
  // ...
]
```

Currently disabled via `showCollectionCategories={false}` in CollectionFilters, but available for future use.

## Best Practices

### Adding New Logomarks

1. Add SVG file to `/packages/ui/src/atoms/logomarks/svg/`
2. Import in `/packages/ui/src/atoms/logomarks/Logomark.jsx`
3. Add entry to `/apps/web/src/data/logomarks.js`:

```javascript
{
  id: 26,
  name: 'Brand Name',
  description: 'Brief description of brand/project',
  logoName: 'brand-name',  // matches SVG filename
  category: 'Brand Identity',
  type: 'Symbol',  // See types list above
  year: '2024'
}
```

### Type Selection Guidelines

- **Primary criterion:** Choose by form first (Symbol, Wordmark, etc.)
- **Secondary:** Industry or style can provide additional context
- **Wordmarks:** Always use `type: 'Wordmark'` for proper sizing
- Be consistent within similar projects
- Update filter options if new types are needed
