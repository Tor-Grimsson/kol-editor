## Troubleshooting

### DNS Not Resolving

**Symptoms**: Can't access kolkrabbi.io
**Causes**:
- DNS still propagating
- Local DNS cache outdated
- ISP DNS slow to update

**Solutions**:
1. Check https://dnschecker.org for global status
2. Try alternative DNS: 8.8.8.8 (Google) or 1.1.1.1 (Cloudflare)
3. Flush local DNS cache (macOS): `sudo dscacheutil -flushcache`
4. Use Vercel URL: kolkrabbi.vercel.app

### SSL Certificate Issues

**Symptoms**: "Not secure" warning, certificate errors
**Causes**:
- Certificate still generating
- Wrong SSL/TLS mode in Cloudflare
- Proxy enabled too soon

**Solutions**:
1. Wait 5-10 minutes after DNS verification
2. Verify Cloudflare SSL/TLS mode: Full (strict)
3. Set proxy to "DNS only" (gray cloud)
4. Check Vercel dashboard for certificate status

### "Too Many Redirects"

**Cause**: SSL/TLS mode mismatch between Cloudflare and Vercel
**Solution**:
1. Cloudflare: SSL/TLS → Full (strict)
2. Clear browser cache
3. Wait 5 minutes
