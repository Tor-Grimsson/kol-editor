# 4.4.3 Specimen Hubs

## Overview

Specimen hubs are landing pages for each typeface's specimen collection. They provide navigation to individual specimen patterns and context about the typeface.

## Hub Structure

### Common Pattern

All specimen hubs follow a consistent structure:

**Location**: `apps/web/src/routes/specimens/[typeface]/[Typeface]Hub.jsx`

**Components**:
1. Back link to specimen library
2. Typeface name and introduction
3. Collection cards for available specimens
4. Routing to individual specimen pages

### Hub Component Pattern

```jsx
import { Link } from 'react-router-dom'
import LinkWithIcon from '@kol/ui'

export default function TypefaceHub() {
  const collections = [
    {
      name: 'Collection Name',
      typeface: 'TG Typeface',
      subtitle: 'Pattern count',
      description: 'Description text',
      link: '/specimen/typeface/pattern',
      category: 'Category'
    }
  ]

  return (
    <main className="min-h-screen w-full bg-surface-primary">
      <LinkWithIcon to="/foundry/specimens">
        Back to All Specimens
      </LinkWithIcon>

      {/* Collection cards */}
      {collections.map(collection => (
        <CollectionCard key={collection.link} {...collection} />
      ))}
    </main>
  )
}
```

## Collection Card Data Structure

```javascript
{
  name: 'Collection Name',
  typeface: 'TG Typeface Name',
  subtitle: '11 Patterns' | 'Complete Set' | 'Coming Soon',
  description: 'Detailed description of what the specimen demonstrates',
  link: '/specimen/typeface/pattern-name',
  category: 'Editorial' | 'Poetry' | 'Systems' | 'Layout' | 'Display' | 'Preview'
}
```

## Hub Navigation Flow

```
Foundry Specimens (/foundry/specimens)
  ↓
Typeface Hub (/specimen/[typeface])
  ↓
Individual Specimen (/specimen/[typeface]/[pattern])
```

**Back Links**:
- Specimen pages → Hub page
- Hub pages → Foundry Specimens

## Styling Guidelines

### Hub Pages
- Background: `bg-surface-primary`
- Max width: `1400px`
- Padding: `px-8 py-16`
- Centered content

### Collection Cards
- Component: Custom per hub or reusable CollectionCard
- Hover states for interactive elements
- Clear typography hierarchy
- Consistent spacing

### Typography
- Hub titles: Large display text in typeface's font
- Descriptions: `kol-mono-text` utility
- Links: LinkWithIcon with arrow
- Category labels: Pill components

## Hub vs Selection Pages

**Hub Page** (e.g., MalromurHub):
- Navigation/landing page
- Shows available specimen collections
- Provides context and overview
- Links to individual specimens

**Selection Page** (e.g., MalromurSelection):
- Actual specimen content
- Combines multiple cards
- Uses GridOverlay
- Full specimen experience
