# Color System - Quick Reference

**Version:** 1.0
**Last Updated:** 2025-11-08
**Status:** Active

> Quick-reference cheat sheet for color tokens and utility classes. For detailed explanations, see [2.1.0 Design System: Color System](./2.1.0-design-system-colors.md).

## Context Utilities

**Tokens:**
| Token | Dark Theme | Light Theme | Usage |
|-------|------------|-------------|-------|
| `--kol-surface-on-primary` | `#fafafa` | `#121215` | Foreground color |
| `--kol-surface-primary` | `#121215` | `#fafafa` | Background color |
| `--kol-border-default` | 8% foreground | 8% foreground | Border color |

**Utility Classes:**
```css
.text-auto              /* color: var(--kol-surface-on-primary) */
.text-auto-inverse      /* color: var(--kol-surface-on-inverse) */

.bg-auto                /* background: var(--kol-surface-primary) */
.bg-auto-inverse        /* background: var(--kol-surface-inverse) */

.border-auto            /* border-color: 8% foreground */
.border-auto-inverse    /* border-color: 8% inverse foreground */

.divider-auto           /* border: 1px, 8% foreground */
.divider-auto-inverse   /* border: 1px, 8% inverse foreground */
```

**Example:**
```jsx
{/* Context utilities set ONE property at a time */}
<div className="bg-auto text-auto border border-auto">
  Each utility sets a single CSS property
</div>
```

## Quick Reference

**Total Utility Classes:** 200+
- Foreground opacity: 39 classes (13 standard + 13 inverse bg + 13 border)
- Opacity-hex: 48 classes (12 standard + 12 inverse × 2 types)
- Opacity-hex-fixed: 48 classes (12 standard + 12 inverse × 2 types)
- Surface: 8 classes
- Container: 3 classes
- Support surfaces: 4 classes
- Context: 8 classes
- State: 15+ classes
- Elevation: 3 classes

**Key Patterns:**
- All utilities adapt to light/dark mode automatically (except fixed)
- Inverse variants use `--kol-surface-on-inverse` as base
- Context-aware utilities work in inverse surface contexts
- State utilities use pseudo-classes (`:hover`, `:active`, `:focus-visible`)
