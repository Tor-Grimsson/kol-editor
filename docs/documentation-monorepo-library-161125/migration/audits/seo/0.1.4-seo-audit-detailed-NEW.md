## 7. TECHNICAL CHALLENGES IDENTIFIED

### SPA Architecture
- React Router DOM client-side routing
- Dynamic page titles need to be set via JavaScript
- Search engines may not execute JS - need SSR or pre-rendering for full SEO benefit
- Dynamic meta tags can't be set in HTML head for SPAs without a library

### CMS Integration
- Some pages fetch content from Sanity CMS (Work projects, Stack articles)
- These need to pull metadata from CMS or generate dynamically
- Currently no mechanism for this

### Multiple Language Support
- App has LanguageContext suggesting bilingual/multi-language support
- Alternate language links not implemented
- Need hreflang tags
