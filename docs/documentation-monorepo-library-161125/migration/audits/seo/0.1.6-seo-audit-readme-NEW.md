## What's Missing

### Critical Gaps (Must Fix):
1. No SEO package installed (need react-helmet-async)
2. No dynamic page titles (all pages show generic "kolkrabbi.io — Web")
3. No meta descriptions (0 pages have descriptions)
4. No OG tags (0 pages have social sharing metadata)
5. No robots.txt
6. No sitemap.xml

### Infrastructure Files Missing:
- robots.txt (file)
- sitemap.xml (file)
- OG image assets (26+ images needed)
- favicon.ico (currently only SVG)

### Phase 1: IMMEDIATE (4-6 hours) - BEFORE LAUNCH
- Install react-helmet-async

### Phase 2: ESSENTIAL (6-8 hours) - LAUNCH WEEK 1
