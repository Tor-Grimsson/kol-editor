## INFRASTRUCTURE REQUIREMENTS

### robots.txt
Create `/public/robots.txt`:
```
User-agent: *
Allow: /
Disallow: /workshop/

Sitemap: https://kolkrabbi.io/sitemap.xml

User-agent: AhrefsBot
Disallow: /
```

Create `/public/sitemap.xml` with:
- All main pages
- All foundry pages
- All collection pages
- All specimen hub pages (32 main hubs)

- Homepage: 1.0
- Main sections (foundry, collections, stack, work): 0.8
- Detail pages: 0.6
- Specimens: 0.5
- Workshop: 0.3 (or excluded if noindex)
