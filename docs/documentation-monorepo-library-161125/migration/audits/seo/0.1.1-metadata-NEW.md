## Required Infrastructure

### 1. robots.txt
**Location:** `/public/robots.txt`

```txt
User-agent: *
Allow: /
Sitemap: https://kolkrabbi.is/sitemap.xml

# Disallow workshop for now (optional)
# Disallow: /workshop/
```

### 2. sitemap.xml
**Location:** `/public/sitemap.xml`
**Method:** Generate programmatically from route config
**Update Frequency:** Weekly or on content changes

### 3. Default OG Image
**Location:** `/public/og-default.png`
**Dimensions:** 1200x630px
**Content:** Kolkrabbi branding + tagline
**Format:** PNG or JPEG (optimized)

### 4. Package Installation
```bash
npm install react-helmet-async
```

## Metadata Templates

### Example: Home Page
```javascript
<Helmet>
  <title>Kolkrabbi — Design Studio & Type Foundry</title>
  <meta name="description" content="Icelandic design studio specializing in typography, branding, and digital design. Explore our type foundry featuring open-source Icelandic typefaces." />
  <meta property="og:title" content="Kolkrabbi — Design Studio & Type Foundry" />
  <meta property="og:description" content="Icelandic design studio specializing in typography, branding, and digital design." />
  <meta property="og:image" content="https://kolkrabbi.is/og-default.png" />
  <meta property="og:type" content="website" />
  <meta property="og:url" content="https://kolkrabbi.is/" />
  <meta name="twitter:card" content="summary_large_image" />
</Helmet>
```

### Example: Typeface Page
```javascript
<Helmet>
  <title>Málrómur — Variable Serif Typeface | Kolkrabbi Foundry</title>
  <meta name="description" content="Málrómur is a variable serif typeface designed for Icelandic editorial and literary design. Free download under SIL Open Font License." />
  <meta property="og:title" content="Málrómur — Variable Serif Typeface" />
  <meta property="og:image" content="https://kolkrabbi.is/og-malromur.png" />
  {/* Additional OG tags */}
</Helmet>
```

## Risk Assessment

### If NOT Implemented Before Launch

**Search Engine Impact:**
- Poor organic visibility for 3-6 months
- Delayed indexing of important pages
- Generic/missing titles in search results
- Recovery time: 6-12 months to regain ranking opportunity

**Social Media Impact:**
- Broken/generic social media previews
- Reduced share engagement
- Unprofessional appearance on Twitter/LinkedIn/Facebook

**Business Impact:**
- Missed traffic at critical launch window
- Reduced typeface discovery
- Lower conversion rates

**Recommendation:** Phase 1 is **NON-NEGOTIABLE** for launch. This is a design/typography studio where visual presentation (including social previews) is core to the brand.
