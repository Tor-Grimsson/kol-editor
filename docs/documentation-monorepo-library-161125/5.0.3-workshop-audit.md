# 5.0.3 Workshop Audit – Layout & Patterns

**Date:** 2025-11-15  
**Status:** Internal Reference  
**Purpose:** Snapshot of current workshop layouts, padding usage, and shared patterns to guide cleanup.

---

## Shell & Navigation
- `WorkshopLayout.jsx` enforces `pt-10 px-4 pb-16 sm:px-8 lg:px-12` (40/16–48/64) plus `space-y-10`.
- `WorkshopSidebar.jsx` consumes `WORKSHOP_ROUTES`, auto-collapsing on sub-pages unless locked; mobile header hosts the theme toggle + dropdown.
- Lazy loading protects the main bundle for chess/analytics routes.

**Gap:** Hall of Mirrors subpages and some apparatus views still embed ad-hoc padding instead of relying on the layout.

---

## Section Review

### Foundational Pages (Colors, Typography, Components)
- Strong use of `DesPage`, `DesSection`, `DesCard`.
- **Gap:** `DesPage.meta` syntax varies (array vs. string); standardize on the string format.

### Apparat / Effects
- `ApparatusCircleGenerator` still wraps everything in `h-full w-full` and handles padding manually.
- `ApparatusImageFilter` now follows the layout baseline and documents controls via `DesSection` + `DesCard`.
- Hall of Displacement/Movement/Copies/Symphony predate `WorkshopLayout`, so their headers/padding differ.

### Chess & Analytics
- Lazy routes with rich documentation cards.
- **Gap:** `AnalyticsComponents.jsx` nests extra `px-4` wrappers that double horizontal padding on wide screens.

---

## Shared Components
- **DesPage** – convert all routes to the new `meta` string format.
- **DesSection** – widely used on Colors/Typography, underused on older apparatus.
- **DesCard** – primary building block; ensure legacy pages adopt it for consistent typography.

---

## Recommended Fixes
1. **Padding Sweep** – Align Hall of Mirrors + apparatus routes with `WorkshopLayout` padding.
2. **Meta Consistency** – Switch all `DesPage` metadata to the `meta` string.
3. **Apparat Naming** – Update 5.3.2–5.3.5 headings/subtitles to say “Workshop” instead of “Components.”
4. **Navigation Docs** – Point 1.5.x references to `5.0.2` (workshop sidebar).
5. **HomeApparat Enhancements** – Surface cards linking to Hall of Mirrors, Frequency Modulator, Image Filter, etc.

Use this audit as the baseline for follow-up tickets and revisit once the fixes land.
