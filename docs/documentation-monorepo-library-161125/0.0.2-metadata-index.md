## How to Use This Index

### For New Contributors

1. **Start with Foundation:**
   - Read [1.0.0 Repository Structure](1.0.0-foundation-repository-structure.md)
   - Review [1.0.1 Naming Conventions](1.0.1-foundation-naming-conventions.md)

2. **Understand the Design System:**
   - Start with [2.0.0 Design System Overview](2.0.0-design-system-overview.md)
   - Learn about [2.1.0 Colors](2.1.0-design-system-colors.md)

3. **Find Documentation:**
   - Use this index to find relevant docs
   - Check "Related Documentation" sections in each doc
   - Use Ctrl+F to search by keyword

### For Documentation Writers

1. **Before Writing:**
   - Read [0.0.1 Writing Guidelines](0.0.1-metadata-writing-guidelines.md)
   - Find the appropriate category
   - Check if an index doc exists for that category

2. **While Writing:**
   - Use correct M.m.p numbering
   - Add frontmatter with metadata
   - Create cross-references to related docs
   - Update this index

3. **After Writing:**
   - Update parent index document
   - Verify all links work
   - Add to this master index

### For Maintainers

- Update this index when adding new documents
- Review quarterly for orphaned documents
- Check cross-reference integrity
- Archive deprecated documents
