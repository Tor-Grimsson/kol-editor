# 4.4.2 Specimens

## Overview

Specimen pages demonstrate typefaces in various contexts and applications. Each specimen showcases specific patterns like editorial layouts, poetry, system typography, or grid systems.

## Specimen Structure

### Specimen Organization

Specimens are organized by typeface in the file structure:
```
apps/web/src/routes/specimens/
├── malromur/
│   ├── cards/              # Individual specimen cards
│   ├── MalromurHub.jsx     # Typeface specimen hub
│   └── MalromurSelection.jsx # Complete specimen selection
├── gullhamrar/
├── dylgjur/
├── silfurbarki/
├── rot/
└── ordspor/
```

### Common Components

**SpecimenHub Pattern**:
Each typeface has a hub page that serves as the landing/navigation page for that typeface's specimens.

**Location Pattern**: `apps/web/src/routes/specimens/[typeface]/[Typeface]Hub.jsx`

**Selection Pattern**:
Most typefaces have a "Selection" or "Complete" page that combines multiple specimen cards into a single continuous page.

**Location Pattern**: `apps/web/src/routes/specimens/[typeface]/[Typeface]Selection.jsx`

### Card-Based Architecture

Specimens are built from reusable card components stored in `/cards/` folders:

**Benefits**:
- Reusable across different specimen pages
- Can be featured in the specimen library carousel
- Easy to maintain and update
- Consistent structure across typefaces

**Card Pattern**:
```jsx
export default function SpecimenCard() {
  return (
    <section className="w-full min-h-screen bg-surface-inverse" style={{...}}>
      {/* Specimen content */}
    </section>
  )
}
```

## GridOverlay Component

**Location**: `apps/web/src/components/specimens/GridOverlay.jsx`

**Purpose**: Provides toggleable grid and baseline overlays for all specimen pages

**Props**:
- `columns` (default: 12) - Number of grid columns
- `gutter` (default: 24) - Gap between columns in pixels
- `marginX` (default: 48) - Horizontal margin in pixels
- `baselineGrid` (default: 24) - Baseline grid spacing in pixels
- `children` - Specimen content to wrap

**Features**:
- Toggleable column grid overlay (blue)
- Toggleable baseline grid overlay (red/gray)
- ViewToggle component with grid icons (grid-04 and grid-02)
- Fixed position toggle button (bottom-right)
- Pointer-events disabled on overlays

**Usage**:
```jsx
import GridOverlay from '../../../components/specimens/GridOverlay'

export default function SpecimenPage() {
  return (
    <GridOverlay columns={12} gutter={24} marginX={48}>
      <SpecimenCard1 />
      <SpecimenCard2 />
    </GridOverlay>
  )
}
```

## Specimen Categories

### Editorial Specimens
**Typefaces**: Málrómur, Dylgjur

**Characteristics**:
- Prose patterns for editorial contexts
- Multiple layout variations
- Running text examples
- Hierarchical typography

**Example**: Málrómur has 11 different prose style patterns

### Poetry Specimens
**Typefaces**: Gullhamrar

**Characteristics**:
- Icelandic poetry examples
- Expressive typography
- Rhythm and spacing emphasis

### Systems Specimens
**Typefaces**: Rót

**Characteristics**:
- Design system typography
- Variable font demonstrations
- Scale and hierarchy systems
- Rest (REST API typography patterns)

### Layout Specimens
**Typefaces**: Orðspor

**Characteristics**:
- Grid system demonstrations
- 12-column layouts
- Editorial layout patterns
- Baseline grid examples

**Files**:
- `LayoutL1.jsx` - 8 grid pattern cards
- `LayoutL2.jsx` - Type size demonstrations
- `LayoutL2_NEW.jsx` - Advanced layouts

### Preview Specimens
**Typefaces**: Silfurbarki, Orðspor, Tröllatunga

**Characteristics**:
- In-development typefaces
- Early previews
- "Coming Soon" status

## Design Tokens

Specimens consistently use design system tokens:

**Backgrounds**:
- `bg-surface-inverse` - Primary specimen background
- `bg-surface-primary` - Hub page backgrounds
- `bg-container-primary` - Card containers

**Text**:
- `text-auto` - Context-aware text color
- `text-fg-64` - Muted text (descriptions)
- `text-fg-48` - Subtle text

**Grid Parameters**:
- Standard 12-column grid
- 24px gutter
- 48px horizontal margins
- 24px baseline grid
