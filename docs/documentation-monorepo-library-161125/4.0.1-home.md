## Quick Reference

- **HomeHero** — “Vinnustofa” headline, GSAP-driven video morph.
- **HomeAbout** — animated atelier intro, floating copy (“Visual identity and type design …”).
- **HomeCardFeatures** — “Featured” cards highlighting Foundry, Client Work, Collections, Workshop.

The home page serves as the primary landing experience, introducing Kolkrabbi studio through a series of carefully orchestrated sections featuring immersive animations, project showcases, and service highlights.

### 1. HomeHero
**File**: `apps/web/src/components/sections/home/HomeHero.jsx`
**Purpose**: Full-viewport hero with animated video background

#### Features
- Full viewport height (dvh) video background
- Theme-aware video switching (dark/light modes)
- GSAP ScrollTrigger animation morphing clip-path and border-radius
- Dual text overlays with blend modes
- Touch device optimization

#### Content
- **Heading**: "Vinnustofa" (Icelandic for "Workshop")
- **Typography**: `kol-heading-display`
- **Font Size**: `clamp(4rem, 8vw, 9rem)` (responsive)

#### Layout
```
Container (h-dvh, w-screen)
├── Video Frame (clip-path animated)
│   ├── Video (theme-dependent)
│   └── Text Overlay (mix-blend-mode: difference)
└── Text Below (absolute positioning)
```

#### Animation
- **Type**: ScrollTrigger-driven morphing
- **Clip-path**: `polygon(14% 0%, 72% 0%, 90% 90%, 0% 100%)` → full rectangle
- **Border-radius**: `0 0 40% 10%` → `0 0 0 0`
- **Timing**: Scrub linked to scroll position
- **Trigger**: Center center → bottom center

#### Props
- `onVideoStart`: Callback when video loads

## Design Patterns

### Container Sizing
All sections use consistent max-width pattern:
```jsx
className="max-w-[1400px] mx-auto"
```

### Typography System
| Class | Usage |
|-------|-------|
| `kol-heading-display` | Hero titles |
| `kol-display-lg` | Section titles |
| `kol-heading-xl` | Card titles (uppercase) |
| `kol-heading-lg` | Section headings |
| `kol-heading-md` | Labels |
| `kol-mono-text` | Body text |
| `kol-mono-sm` | Supporting text |
| `kol-mono-xs` | Small text/labels |
| `text-auto` | Theme-aware coloring |

### Responsive Breakpoints
- **Mobile**: Default (flex-col, stacked)
- **md**: 768px (horizontal layouts begin)
- **lg**: 1024px (full desktop layouts)

### Animation Libraries
- **GSAP**: ScrollTrigger for scroll-based animations (HomeHero, HomeAbout)
- **Framer Motion**: Alternative for advanced effects (HomeHighlights optional)
- **Intersection Observer**: Visibility-based triggers (HomeSignup)
- **Custom Hooks**: `useBentoTilt`, `useBentoTiltMotion`, `useIsTouchDevice`

### Touch Device Handling
- Detection via `useIsTouchDevice` hook
- Animations disabled/modified on touch devices
- Video pointer events disabled on mobile (HomeHero)
- Tilt effects disabled (HomeHighlights)

## Performance Considerations

### Video Optimization
- Autoplay, loop, muted attributes for silent autoplay
- Preload metadata only
- Theme-based video switching
- Touch device optimizations

### Animation Performance
- CSS transforms for hardware acceleration
- `backface-visibility: hidden` to prevent flickering
- Scroll-linked animations use GSAP's efficient scrub
- Intersection Observer for visibility-based triggers (not scroll listeners)

### Image Loading
- Static images via `/img` directory
- Consider lazy loading for below-fold images
- Aspect ratio preservation to prevent layout shift

Each section is carefully crafted to guide users through the Kolkrabbi experience while maintaining visual consistency and brand identity.
