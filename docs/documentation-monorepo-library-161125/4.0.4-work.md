# 4.0.4 Work Page

**Route:** `/work`  
**Component:** `apps/web/src/routes/Work.jsx`  
**Status:** Active

---

## Mission

Showcase shipped projects, surface featured case studies, and drive inquiries.

---

## Data & State

- `getAllProjects()` fetches published Sanity `project` docs.
- `useEffect` guards against state updates after unmount.
- Results feed both grid + list views; `WorkHeroSection` reads the same array.

---

### Section Notes
- **Hero:** Key stats + featured clients (`WorkHeroSection`).
- **ImageSection:** Visual spacer featuring hero art.
- **ProjectsGrid:** Masonry-style cards with thumbnail + metadata.
- **ProjectsList:** Detailed rows for deep context (services, timeframe).
- **CTA:** Shared `CtaGlobal`.

---

## Content Requirements

- Each project needs `thumbnail`, `description`, `services`, `year`. Missing data will render blank states.
- Set `featured: true` in Sanity to ensure hero section has content.
