## Routing

The workshop router mounts everything behind `/workshop/*` with lazy-loaded bundles to avoid bloating the marketing site.

```tsx
// apps/web/src/routes/Workshop.jsx
<Routes>
  <Route path="analytics/*" element={<AnalyticsRouter />} />
  <Route path="chess/*" element={<ChessRouter />} />
  <Route path="effects/*" element={<EffectsRouter />} />
  <Route path="apparatus/*" element={<ApparatusRouter />} />
</Routes>
```

Shared patterns:
- `DesPage` header wrapper
- DesSection
- DesCard
-----Des* don't accept components these are labels

- `WorkshopSidebar` for intra-section navigation
- `WorkshopCard` grid for apparatus previews
